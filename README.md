# NDVI Spatial Mapping and Vegetation Health Analysis of Dhaka Division

# Project Overview
This repository contains the spatial analysis and mapping of the Normalized Difference Vegetation Index (NDVI) for the Dhaka Division, Bangladesh. The primary focus of this project is to assess vegetation health, monitor environmental conditions, and identify areas under ecological stress using remote sensing techniques. 

# Map Classification & Output
The analysis categorizes the NDVI values into five distinct classes to visualize the vegetation status effectively:

**Severe Stress:** Indicated by Dark Green
**Moderate:** Indicated by Light Green
**Fair:** Indicated by Yellow
**Good:** Indicated by Orange
**Excellent:** Indicated by Red

# Technologies & Tools Used
**GIS Software:** ArcGIS (for spatial analysis, map composition, and layout creation)
**Methodology:** Remote Sensing Index Calculation (NDVI)
**Domain:** Environmental Science, Geographic Information Systems (GIS), Disaster Risk Reduction

# Key Objectives
*   To calculate and map the NDVI values across Dhaka Division using satellite imagery.
*   To visually classify vegetation health to assist in environmental monitoring, agricultural planning, and resource management.
*   To create a professional, publication-ready cartographic layout with proper grid coordinates, scale, and legends.

# Future Scope
*   Integration of multi-temporal satellite data to monitor seasonal vegetation changes.
*   Correlation of NDVI data with local climatic factors (temperature, rainfall).
